# Example Golang
Dockerized (PostgreSQL + golang + redis)

## Features
 - Uses multi-stage builds for small production images 
 - Uses .env file

## Requirements
 - Install docker && docker-compose

## Quick Start
1. `docker-compose up -d`
2. localhost:8080/get or localhost:8000/set for use Redis
